Next.js 是一个强大的 React 框架，让我们能轻松构建全栈 Web 应用。

## 主要特性

- **App Router** - 现代化的路由系统
- **Server Components** - 服务端组件提升性能
- **API Routes** - 轻松创建 API 接口
- **自动优化** - 图片、字体等自动优化

## 创建第一个页面

```tsx
export default function Home() {
  return <h1>Hello, Next.js!</h1>;
}
```
